Summary

Abstract

Recent advances in large language models enable robots to generate task plans from natural language instructions, but converting these plans into reliable structures for long-horizon tasks remains challenging. This work compares LLM-based Behavior Tree generation methods and proposes a backward BT expansion method that generates subtrees from failed or goal conditions.

The proposed method uses the relationship among failed conditions, preconditions, and actions to structurally connect required actions with their supporting conditions. LLM-COBRA combines high-level goal decomposition with low-level backward chaining and reuses stored PPA entries to reduce repeated LLM queries and token usage.

Method

Framework

LLM-COBRA first translates a user instruction into goal conditions, then expands failed conditions through a low-level planner that generates and reuses PPA relations.

High-Level Planner

Transforms natural language instructions into goal conditions and initial BT structure.

Low-Level Planner

Uses condition-driven backward chaining to create reactive fallback subtrees.

PPA Library

Stores failed condition, precondition, and action relations for reuse across trials.
